Этот проект демонстрирует автоматизацию развертывания инфраструктуры в Yandex Cloud с использованием **Terraform** и **Ansible**.
В этом репозитории вы найдете:
- **Terraform**: создание ВМ в Yandex Cloud
- **Ansible**: настройка серверов, установка Nginx, прокси и отдача страницы
- **Роли Ansible**: модульная структура для простоты повторного использования
